Work Stocks HTML Job Board Template Documentation

1. Introduction

Work Stocks is an alluring, advance, fully responsive, gorgeous and full feature packed HTML Job board theme for your next project. This theme is complied of the most popular job manager plugin, WP Job Manager that offers an extremely website creation experience.


We put a lot of love and effort to make Work Stocks a useful template for everyone and now It comes with 8+ unique home page & colors. We are keen to release continuous long term updates and lots of new features will be coming soon in the future releases. Grid classes apply to devices with screen widths greater than or equal to the breakpoint sizes, and override grid classes targeted at smaller devices. Therefore, applying any .col-md- class to an element will not only affect its styling on medium devices but also on large devices if a .col-lg- class is not present.
